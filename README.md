
# BillSense AI

AI-powered utility bill intelligence platform for electricity and gas consumers.

## Features
- OCR bill extraction
- AI bill explanation
- Bill forecasting
- Smart savings recommendations
- AI dispute assistant
- MSME analytics dashboard

## Stack
- Next.js
- TypeScript
- TailwindCSS
- FastAPI
- LangChain Agents
- OpenAI API
- Scikit-learn
- PostgreSQL

## Deployment
Frontend → Vercel  
Backend → Render/Railway

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```
